## рҹҺү What is CultureHub?

CultureHub is a **fully self-contained, zero-dependency** office culture event tracker built with plain HTML, CSS, and JavaScript. Drop it on a file server, a shared drive, or just open it locally вҖ” it works everywhere, stores everything in your browser's `localStorage`, and looks sharp doing it.

Built for teams that want to **track, celebrate, and reward** the people who show up, participate, and bring the energy.

> **TL;DR:** Think trivia nights, bowling leagues, holiday parties, team-building events вҖ” CultureHub tracks who came, who won, and who's climbing the leaderboard. рҹҸҶ

---

## вңЁ Features

### рҹ“… Event Management
Create and manage any type of team event with full metadata вҖ” title, date, type, location, and description. Filter by type, search by name, and jump straight to check-in or gallery from every event card.

```
Event Types: General В· Game В· Social В· Team Building В· Holiday В· Charity
```

### рҹ‘Ҙ Roster Management
Maintain a living roster of your team with names, departments, and emails. Import in bulk from a JSON file вҖ” including **Microsoft Teams / Azure AD export format** (`displayName`, `mail`, `department`). No manual entry required.

### вң… Live Check-In
The heart of CultureHub. Select an event, tap a card to check someone in, tap рҹҸҶ to crown a winner.

```
вң… Attendance  вҶ’  1 point
рҹҸҶ Game Winner вҶ’  3 points
```

Check-in is instant, visual, and satisfying. Every card lights up green when checked in and gold for winners.

### рҹ“Ҡ TV Dashboard / Leaderboard
A **full-screen TV mode** built for the break room display. Hit `рҹ“ә TV Mode` and it goes fullscreen вҖ” animated leaderboard, live ticker tape, attendance bars, auto-refreshes every 30 seconds. Press `Esc` to exit.

### рҹ”Қ Search
Search by **person** or **event** with instant side-by-side detail panels. See every event a person attended, every person who attended an event, win/loss records, and points breakdowns вҖ” all in one place.

### рҹ–ј Photo Gallery
Every event gets its own photo gallery with drag-and-drop upload, captions, and a **fullscreen slideshow** with keyboard navigation (`вҶҗ` `вҶ’` `Esc`). Because memories matter.

### рҹ“„ PDF Reports
One-click print-ready PDF reports for:
- рҹ“Ҡ **Overview** вҖ” org-wide leaderboard, stats, all events
- рҹ“… **Per Event** вҖ” full attendance list with points
- рҹ‘Ө **Per Person** вҖ” individual history and score breakdown

### рҹ’ҫ Backup & Restore
Full JSON backup/restore вҖ” download your data anytime, restore from any backup file. Your data lives in `localStorage` but you own it completely.

### Key Design Decisions

| Decision | Why |
|----------|-----|
| **Zero dependencies** | No node_modules, no CDN failures, no version conflicts |
| **localStorage** | Works offline, no backend needed, data stays with the user |
| **Multi-file HTML** | Each page is independently navigable, easy to maintain |
| **Shared `data.js`** | Single source of truth вҖ” all pages read/write the same store |
| **CSS variables** | Full dark/light theme switching with zero JS class toggling |

## рҹҺЁ Design System

CultureHub uses a custom design system with:

- **Fonts:** [Outfit](https://fonts.google.com/specimen/Outfit) (headings) + [Plus Jakarta Sans](https://fonts.google.com/specimen/Plus+Jakarta+Sans) (body)
- **Colors:** Teal accent В· Coral red В· Amber gold В· Navy canvas
- **Themes:** Full dark and light mode вҖ” toggle in the nav, persisted to localStorage
- **No flash:** Inline script in `<head>` applies saved theme before first paint

```css
/* Teal accent (dark mode) */
--accent: #00c9b1;

/* Same accent adapted for light mode contrast */
--accent: #007f6f;
```

## рҹ“Ҙ Importing Team Members

CultureHub supports bulk import from a JSON array. Works with:

**Simple format:**
```json
[
  { "name": "Alice Johnson", "team": "Engineering", "email": "alice@company.com" },
  { "name": "Bob Smith",     "team": "Marketing",   "email": "bob@company.com" }
]
```

**Microsoft Teams / Azure AD export format:**
```json
[
  { "displayName": "Alice Johnson", "mail": "alice@company.com", "department": "Engineering" },
  { "displayName": "Bob Smith",     "mail": "bob@company.com",   "department": "Marketing" }
]
```

Import via **Roster вҶ’ рҹ“Ҙ Import** or **Settings вҶ’ Import from Team**.

## рҹ’ҫ Data & Privacy

All data is stored **100% locally** in your browser's `localStorage` under the key `culturehub_data`. Nothing is ever sent to a server.

```
localStorage['culturehub_data'] = {
  version: 1,
  people: [ { id, name, team, email } ],
  events: [ { id, title, date, type, location, description, gallery, attendees } ],
  settings: { orgName, theme }
}
```

**Backup regularly** вҖ” `localStorage` can be cleared by browser settings. Use **Settings вҶ’ рҹ’ҫ Download Backup JSON** to export, and **Settings вҶ’ вҷ»пёҸ Restore** to import.
